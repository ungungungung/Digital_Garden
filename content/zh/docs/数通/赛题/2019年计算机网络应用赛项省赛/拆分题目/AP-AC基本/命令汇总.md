---
title: 命令汇总
---
```
# 显示AP上线情况
show ap-config summary

# 进入AP
ap-config AP3

# 进入AP后配置AP名称
AC1(config-ap)#ap-name AP1

wlan-config wlan_id ssid_name

interface-mapping wlan_id vlan_id

# AP加入AP组
Ruijie(config-ap)#ap-group ap_group_name

# 本地转发和集中转发
tunnel local
tunnel 8023

# ac指定ap恢复出厂设置
ac-controller
factory-reset ap_name

# console线进入ap恢复出厂设置
ap(config)#apm factory-reset

# 配置wifi密码(WPA2)
wlansec wlan_id
security rsn enable
security rsn ciphers aes enable
security rsn akm psk enable
security rsn akm psk set-key ascii p0-p0-p0-
exit

# 配置capwap隧道地址
ac-controller    
capwap ctrl-ip 192.168.1.1

# 查看AC相关信息
show ac-config

# 查看ap组信息
show ap-group aps summary 

show ap-group intf-wlan-map FB

```
